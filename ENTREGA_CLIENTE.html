📋 ¿Qué es este sistema?

El Sistema de Gestión Dulce Guerrero'z es una aplicación web desarrollada a la medida para administrar de forma digital el inventario de insumos, las recetas de productos, el registro de ventas y el control financiero del negocio.

Funciona desde cualquier navegador web — computador, tableta o celular — sin necesidad de instalar ningún programa. Basta con tener internet y abrir el enlace proporcionado.

✅

Acceso inmediato desde el celular

El sistema está diseñado pensando en el uso móvil: menú adaptable, botones grandes, tablas con información prioritaria y navegación sencilla.

📖 Tutorial de uso

Para el administrador

1

Iniciar sesión

Ingresa al enlace del sistema, escribe tu correo electrónico y contraseña. El sistema te llevará directamente al dashboard con el resumen del día.

2

Configurar insumos

Ve a 🧂 Insumos → + Nuevo insumo. Ingresa el nombre, unidad de medida, stock inicial y stock mínimo. El stock mínimo genera alertas automáticas cuando el inventario está bajo.

3

Crear recetas y productos

Ve a 📝 Recetas → Crear receta. Escribe el nombre del producto, su precio de venta, y selecciona los insumos con la cantidad que consume cada unidad producida. Esto es lo que el sistema descuenta automáticamente al vender.

4

Registrar entradas de stock

Cada vez que llegue mercancía, ve a 📦 Entradas → Registrar entrada. Selecciona el insumo, la cantidad recibida y la fecha. El stock se actualiza solo.

5

Registrar gastos e ingresos

En 📊 Finanzas puedes registrar gastos (servicios, personal, materias primas, mantenimiento) e ingresos manuales. Las gráficas se actualizan automáticamente mostrando el balance del período.

6

Crear usuarios para empleados

Ve a 👥 Usuarios → + Nuevo usuario. Ingresa nombre, correo y contraseña del empleado. Asigna el rol "Empleado" para limitar su acceso solo a ventas e inventario.

Para el empleado

1

Iniciar sesión y revisar el dashboard

Al entrar verás cuántas ventas se han hecho hoy y si hay insumos con stock bajo. Es la primera pantalla que debes revisar al comenzar el turno.

2

Registrar una venta

Toca 💰 Nueva venta. Busca y selecciona el producto vendido, ingresa la cantidad y toca Confirmar venta. Si el cliente compró varios productos distintos, usa el botón "+ Agregar otro producto" antes de confirmar.

3

Hacer conteo de inventario

Ve a 📋 Hacer conteo. Usa el buscador para encontrar rápidamente cada insumo, escribe la cantidad real que encontraste en bodega y al terminar presiona Guardar conteo. Solo los campos que llenes serán actualizados.

4

Consultar el inventario

En 🧂 Ver inventario puedes ver el stock actual de todos los insumos. Los que tienen fondo rojo están por debajo del mínimo y necesitan reposición.
